Granta

Material Selection

  1. Search Remote Desktop Connection and login to Ichat email
    Database to be selected is Advanced level 3
  2. Launch Granta and start project
  3. Select material database
  4. Generate chart by selecting X and Y axes value.
  5. Generate a line with slope value
    slope value to be calculate using table of performance index (can get in Granta Help).
  6. Repeat for another chart with different Y-axis value
    7. Use a box instead of line when in the graph of price
  7. Limit the material to 5-10 by dragging the line/box in each graph to certain position.
